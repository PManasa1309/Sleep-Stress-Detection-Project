# 🧠 Sleep Stress Detection using Machine Learning

## 📌 Overview

This project focuses on detecting **stress levels during sleep** using Machine Learning algorithms.

It compares multiple algorithms to find the most accurate model for stress prediction.

---

## 🚀 Algorithms Used

* 🌲 Random Forest (Best Model)
* 🌳 Decision Tree
* 📈 Support Vector Machine (SVM)
* 📍 K-Nearest Neighbors (KNN)
* 📊 Logistic Regression

---

## 📊 Results

| Algorithm           | Accuracy  |
| ------------------- | --------- |
| Random Forest       | **85.5%** |
| KNN                 | 82.6%     |
| SVM                 | 79.5%     |
| Logistic Regression | 75.6%     |
| Decision Tree       | 71.4%     |
